## 📋 术语翻译原则

1. **保持一致性** - 相同术语在整应用中应使用相同翻译
2. **简洁明了** - 使用简洁的中文表达，避免冗长
3. **符合中文习惯** - 调整语序以符合中文表达习惯
4. **技术准确性** - 保持技术概念的准确性

## ⚠️ 注意事项

1. **Session** 在不同语境下可能有不同的翻译：
   - UI 列表中：会话
   - 文件名中：Session
   - 技术说明中：会话

2. **Resume** 根据上下文选择翻译：
   - 按钮文字：继续
   - 操作说明：恢复/继续执行
   - 文件名：Resume

3. **Project** 根据语境区分：
   - 工作项目：项目
   - Xcode 项目：Project（保持原样）
   - 项目配置：项目配置

4. **Terminal** 的翻译：
   - macOS 终端：终端
   - 嵌入式终端：内嵌终端/嵌入式终端
   - iTerm2：保持原名 iTerm2
   - Warp：保持原名 Warp
